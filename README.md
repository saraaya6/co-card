# Co-Card

## وصف المشروع

هذا المشروع هو موقع ويب بسيط يعرض بطاقة لمجموعة تقنية تسمى "Bit to Best" (BTB). يحتوي الموقع على معلومات عن الأعضاء، التخصصات التقنية، وطرق الاتصال.

## الميزات

- عرض شعار المجموعة
- قائمة بالتخصصات التقنية المتاحة (AI، Full Stack Web، Mobile Development، UI/UX، Cyber، Networking)
- جدول بأعضاء المجموعة مع أعمارهم وتخصصاتهم
- معلومات الاتصال
- رابط للانضمام عبر Telegram

## كيفية التشغيل

1. قم بتنزيل الملفات إلى جهازك.
2. افتح ملف `index.html` في متصفح الويب.
3. الموقع جاهز للعرض!

## المتطلبات

- متصفح ويب حديث (مثل Chrome، Firefox، Safari)

## هيكل المشروع

- `index.html`: الملف الرئيسي للموقع
- `style.css`: ملف التصميم
- `images/`: مجلد الصور
- `activity_log.txt`: سجل النشاط
- `contribution_log.txt`: سجل المساهمات
.
